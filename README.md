# Camel App

### Description

Introduction to the Camel framework.

### Tech Stack
- Spring Boot (JDK 11)
- Camel
- Maven

### Setup & Launch

#### Maven Launch
1. `mvn spring-boot:run`
2. Open browser at http://localhost:8080
